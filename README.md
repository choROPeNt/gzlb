## UE6 - Stoffgesetz und Festigkeit

Für die Übungen UE6-1 bis UE6-5 stehen begleitende Jupyter-Notebooks zur Verfügung, die die Bearbeitung der Aufgaben unterstützen.
Die Notebooks können auf [Google Colab](https://colab.research.google.com/) ausgeführt werden, sodass keine lokale Installation von Python erforderlich ist.
